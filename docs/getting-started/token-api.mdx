description: "Create your own blockchain tokens (ERC-20, ERC-721, ERC-1155) and automate their distribution through one API."

Tokens are the fundamental data model for attributing value and annotating addresses with
decentralized data. There are 3 kinds of offerings, each tailored to serve different kinds of
use cases. To get started, learn more about a specific token standard through its recommended use cases.
All token options support custom transferability constraints and evolving metadata.
