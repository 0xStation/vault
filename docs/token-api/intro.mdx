Station provides token infrastructure for decentralized networks. With Station, you can programmatically
mint and distribute NFTs and ERC-20s to your users and contributors based on customized logic.

### Why Station?

- **For developers:** Set up digital-native incentive programs in minutes with developer-friendly APIs.
- **For community managers:** Make better decisions with a visualization of token ownership and network activities.
