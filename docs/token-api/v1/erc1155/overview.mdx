Using the NFT contract address, you can [update badge metadata](/token-api/v1/erc1155/put/metadata) to prepare
the content of the NFT prior to minting it to your network.

Using the NFT contract address, you can [mint tokens to your network](/token-api/v1/erc1155/post/mint).
Our blockchain indexing systems will automatically pick up these operations and organize the activity.
