To simplify NFT metadata management, Station supports hosting for your assets.
Note that media will be not be transformed or compressed in our systems so take
dilligence to find your preferable balance of file size contrasting quality and load speeds.
To get an API key to start using media hosting, [start here](https://calendly.com/symmtry/chat)
by booking a short onboarding call so we can evaluate your project's needs and pricing.

Read more in the [Upload Media](/token-api/v1/upload) page.
